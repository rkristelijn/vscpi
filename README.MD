# Visual Studio Code on raspberry pi

Use these steps to start working with JavaScript on and from the raspberry pi

I wanted to document every little step, but it seems it was already done for me:

You need only these two steps:

1. Become super user: `sudo -s`
2. Download and run script: `. <( wget -O - https://code.headmelted.com/installers/apt.sh )`

* https://code.headmelted.com/
* https://github.com/Microsoft/vscode/wiki/How-to-Contribute#build-and-run-from-source
* http://www.hanselman.com/blog/BuildingVisualStudioCodeOnARaspberryPi3.aspx
